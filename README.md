# VANTA

Official landing page for **VANTA** — a premium men's stainless steel jewelry brand based in Karachi, Pakistan.

VANTA is built around minimal design, timeless aesthetics, and high-quality stainless steel jewelry inspired by modern streetwear.

---

## ✨ Features

- Premium minimal design
- Mobile-first & responsive
- Product showcase
- WhatsApp Business integration
- Instagram integration
- QR-code friendly
- Fast loading
- Hosted on GitHub Pages

## 🎯 Purpose

The VANTA landing page provides customers with a seamless way to:

- Explore the latest jewelry collection
- Connect instantly via WhatsApp Business
- Visit our Instagram page
- Discover upcoming product drops
- Access VANTA through QR codes on business cards, packaging, and event stalls

## 🌟 Why VANTA?

At VANTA, we believe accessories should be simple, timeless, and built to last.

Every piece is carefully selected to complement modern streetwear while maintaining a clean and minimal aesthetic.
